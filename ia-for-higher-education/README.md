# Higher Education IA Planning Assistant

A research-backed Custom GPT tool for developing information architecture plans 
for academic websites, with specialized guidance for institutional, departmental, 
and service provider sites.

**Created by Doug Worsham in collaboration with Claude (Anthropic)**  
**Version:** 3.0  
**Last Updated:** November 6, 2025

## What This Is

An AI assistant that helps web professionals and student employees design 
effective information architecture for higher education websites. Built on 
comprehensive research from Nielsen Norman Group, WCAG accessibility standards, 
and SEO best practices.

## Key Features

- Site-type-specific "Top 4 Questions" frameworks
- Research-backed navigation principles (debunks the 3-click rule myth)
- SEO and IA integration guidance
- Acronym handling strategies for higher ed jargon
- Current IA analysis methodology for redesigns
- WCAG 2.1 Level AA accessibility compliance
